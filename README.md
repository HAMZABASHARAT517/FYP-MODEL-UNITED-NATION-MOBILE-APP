Overview
Prep-MUN is a mobile application designed to streamline the **Model United Nations (MUN)** experience for delegates, organizers, and institutions. The app provides a complete end-to-end solution for conference management, delegate registration, payments, session tracking, and community engagement.

This project showcases both **mobile app development skills and database management expertise**, combining Flutter & Dart with MySQL/Firebase backend integration.

---

Project Goals
- Develop a comprehensive mobile application for MUN conferences.
- Automate registration, profile management, and payment collection.
- Provide a community platform for networking and communication among delegates.
- Enable conference management for schedules, committees, and sessions.
- Ensure secure and efficient storage of participant data using relational and cloud databases.

---

Key Features
- AI-powered Chatbot to guide delegates and answer FAQs
- Delegate Registration & Profile Management
- Secure Payment Gateway integration for conference fees
- Conference & Committee Management for schedules and sessions
- Community Chat & Networking Hub
- Database Management for storing and analyzing participant information

---

Challenges & Solutions
- Challenge: Designing an intuitive UI/UX for first-time users
  Solution: Conducted user research and applied best practices in mobile app design
- Challenge: Managing secure payment integration
  Solution: Implemented Stripe API with encrypted transactions
- Challenge: Synchronizing cloud databases with real-time app data
  Solution: Integrated Firebase for real-time updates and efficient cloud storage
